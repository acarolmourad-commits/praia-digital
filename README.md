# Praia Digital

Tecnologia especializada no mercado imobiliário do litoral.

## Sobre
Plataforma digital para imóveis de praia — gestão, busca e divulgação de propriedades no litoral brasileiro.

## Stack
- Frontend: HTML5 + CSS3 + JavaScript (Vanilla)
- Deploy: GitHub Pages (github.com/acarolmourad-commits/praia-digital)
