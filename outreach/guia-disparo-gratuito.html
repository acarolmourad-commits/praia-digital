Guia de Disparo Gratuito — Outreach Praia Digital

Opção 1 — Mail Merge para Gmail

Use os arquivos CSV prontos em outreach/importacao-email.csv e outreach/importacao-followups.csv.

  1. Instale a extensão Mail Merge no Chrome.
  2. Abra o Gmail e crie um rascunho com o texto do e-mail.
  3. No planilha Google, faça upload do CSV e use colunas {{email}}, {{subject}}.
  4. Dis pare até 50 e-mails/dia gratuitamente.

Opção 2 — Outlook Desktop

Importe o CSV no Outlook e use o Mail Merge nativo com o template em outreach/kit-comercial-completo.html.

Opção 3 — Script Python local

Configure SMTP_USER e SMTP_PASSWORD no arquivo scripts/email_batch_sender.py e execute:

python scripts/email_batch_sender.py
