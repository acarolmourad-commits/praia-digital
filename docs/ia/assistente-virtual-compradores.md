# Assistente virtual para compradores

## O que é
Chatbot especializado em dúvidas de compra de imóveis no litoral.

## Benefícios
- Atendimento 24h
- Qualificação rápida
- Encaminhamento para WhatsApp

## Fluxo
1) Saudação e cidade de interesse
2) Perguntas qualificadoras
3) Apresentação de imóveis sugeridos
4) Agendamento ou encaminhamento

## Copy de venda
"Atenda todos os compradores, mesmo fora do horário."
