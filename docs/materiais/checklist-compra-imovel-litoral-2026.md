# Checklist de compra de imóvel no litoral
- [ ] Defina orçamento total
- [ ] Escolha cidade e bairro
- [ ] Visite pelo menos 5 imóveis
- [ ] Verifique documentação do imóvel
- [ ] Simule financiamento ou consórcio
- [ ) Confira custos de condomínio e IPTU
- [ ) Avalie estado de conservação
- [ ) Antecipe negociação e prazo
