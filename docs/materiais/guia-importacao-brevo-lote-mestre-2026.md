# Guia rápido de importação no Brevo — lote mestre Praia Digital
1. Acessar contatos > importar > CSV
2. Selecionar `csv-lotes-email/lote-mestre-unificado-2026-07-10.csv`
3. Mapear colunas: nome, email, telefone, cidade, assunto, mensagem
4. Criar lista `Parcerias Litoral 2026`
5. Ativar double opt-in se necessário
6. Enviar e-mail de teste antes do lote completo
