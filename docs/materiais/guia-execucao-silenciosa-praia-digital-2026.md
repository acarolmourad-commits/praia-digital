# Guia de Execução Silenciosa — Praia Digital

Este guia permite executar a rotina completa sem conhecimento técnico.

## Como funciona
1. Clique duas vezes no arquivo `.bat` correspondente.
2. O script roda em segundo plano e gera relatórios automaticamente.
3. Abra o painel diário para ver os resultados.

## Scripts disponíveis
- `scripts/executar_prospeccao_diaria.bat` — abre todos os templates e checklists do dia
- `scripts/executar_followup_diario.bat` — processa follow-ups automáticos e gera relatório
- `scripts/executar_relatorio_diario.bat` — gera o relatório diário de prospecção em HTML
- `scripts/executar_advance_negociacao.bat` — classifica leads e sugere próxima ação comercial
- `scripts/executar_sitemap.bat` — atualiza o sitemap.xml do site

## Passo a passo para usuário não técnico
1. Acesse a pasta `scripts` no repositório.
2. Clique duas vezes no script desejado.
3. Aguarde o fim da execução.
4. Abra o arquivo HTML de saída para ver o resultado.
