# Checklist de captação de imóveis de temporada — corretores do litoral
- [ ] Defina o perfil do proprietário: investidor ou usuário
- [ ] Liste diferenciais relevantes para temporada
- [ ] Registre dados de ocupação e sazonalidade
- [ ] Documente fotos e vídeos atualizados
- [ ] Apresente dados de retorno estimado
- [ ] Proponha avaliação gratuita ou prévia
- [ ] Agende follow-up em até 72h
