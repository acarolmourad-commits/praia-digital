# Pacote de envio real — leads reais

## Como usar
1. Abra a página do lead em `outreach/prospeccao-lead-<id>-<cidade>.html`
2. Copie o e-mail sugerido
3. Envie pelo seu e-mail comercial
4. Marque no tracker `docs/sales/tracker_envios_personalizado.csv`
