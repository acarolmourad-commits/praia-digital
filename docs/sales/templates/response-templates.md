# Templates de Resposta — Negociação B2B

Use essas respostas para avançar rapidamente quando o cliente demonstrar interesse, pedir preço, prazos ou condições especiais.

## Se o cliente perguntar preço

"Para cada caso fazemos uma proposta alinhada ao tamanho da carteira e ao objetivo (captação de leads, SEO, conteúdo recorrente). Prepacotei uma faixa de investimento que costuma caber em imobiliárias de médio porte no litoral. Se quiser, envio já a proposta customizada."

## Se pedir garantia / risco zero

"Sim, fazemos um piloto de 30 dias sem fidelidade. Se não perceber valor, não continuamos. Para nós, o risco é baixo porque a entrega é concentrada nos primeiros 30 dias."

## Se questionar integração técnica

"A integração é simples: botão flutuante no site ou widget embarcado, sem necessidade de alterar layout atual. Se for WordPress, Toptal ou site estático, o tempo médio cai para menos de 1 semana."

## Se pedir cases / referências

"Até o momento nossos cases são plataformas internas, mas a arquitetura do Praia Digital já está validada por usuários reais. Podemos incluir relatórios de desempenho no piloto para mensurar resultados."

## Se pedir desconto

"Podemos estruturar condições especiais para primeiro mês ou para indicação de outras imobiliárias da região. Isso costuma ser mais vantajoso do que desconto puro, porque você tem um custo menor e nós expandimos a rede."

## Se pedir customização / feature adicional

"Vamos priorizar entregas de maior impacto para sua operação no primeiro mês; features extras entram em sprints seguintes sem comprometer o prazo de lançamento."
