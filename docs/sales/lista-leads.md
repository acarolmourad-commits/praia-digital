# Lista de leads para prospecção

## Como usar
Preencha essa lista antes de cada campanha. Mantenha os dados das imobiliárias em planilha offline; este arquivo serve como molde operacional.

## Colunas sugeridas
- Nome da imobiliária
- Site
- Cidade
- Pessoa de contato
- Cargo
- E-mail
- WhatsApp
- Canal preferido
- Status
- Observações

## Critérios rápidos de triagem
1. Tem site ou perfil ativo
2. Tem anúncios em portais/google me negócio
3. Não usa chatbot/IA atualmente
4. Público-alvo inclui imóveis de temporada ou investidores

## Status
- Novo: não contatado
- Contact: e-mail enviado
- Replied: respondeu
- Meeting: reunião agendada
- Pilot: piloto iniciado
- Won: contratado
- Lost: não interessado
- Paused: sem retorno após 3 follow-ups
