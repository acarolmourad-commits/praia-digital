# Rotina Mestre Unificada
## Arquivo principal: scripts/rotina-mestre-praia-digital.bat

### Passos
1. Backup incremental da base comercial
2. Geração do briefing dinâmico
3. Atualização do tracker de envios
4. Abertura da central de comando e arquivos do dia
5. Checklist diário

### Uso
- Duplo clique no arquivo `.bat`
- Ou execute via terminal: `scripts/rotina-mestre-praia-digital.bat`

### Observações
- Requer Python instalado
- Requer configuração SMTP no script `enviar_emails_automatico.py`
- Mantenha o tracker sempre atualizado após envios
